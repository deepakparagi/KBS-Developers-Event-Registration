# KBS Developers Event Registration

A premium, interactive event registration landing page built with Next.js, React, Tailwind CSS, and Framer Motion. Featuring hyper-realistic architectural photography, physics-based "Apple-style" motion design, and a luxury aesthetic.

## Quick Deploy

Deploy this project immediately to Vercel using the button below:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fdeepakparagi%2FKBS-Developers-Event-Registration)

## Tech Stack
- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- Framer Motion
- TypeScript

## Local Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables (if applicable, e.g., for Google Sheets API integration in `.env.local`):
   ```env
   GOOGLE_SHEETS_PRIVATE_KEY="your_private_key"
   GOOGLE_SHEETS_CLIENT_EMAIL="your_client_email"
   GOOGLE_SHEET_ID="your_sheet_id"
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
